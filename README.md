# CAD-keychain

Build Guild Bhopal Event: Keychain Project
**Event Overview**
I attended the Build Guild Bhopal event where I participated in a hands-on workshop focused on CAD software 
Event Rating: (10/10)

Objective: To design and prepare a custom keychain using professional design tools.

**Tools & Technologies**
CAD Software: ONSHAPE

**What I Learned**
This session was a deep dive into CAD using Onshape. I started with a blank canvas and navigated the 3D workspace to model a custom keychain from the ground up. Along the way, I had to solve real-world design puzzles—like making sure the hole for the ring was the right size (tolerances!) and ensuring the whole piece was structurally sound. I finished by prepping the files for production, feeling much more confident in my end-to-end design workflow. 

**Future Improvements**
Now that the basic model is complete, I plan to experiment with more complex geometries and perhaps add embossed text or custom logos in the next iteration.
